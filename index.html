使用文档

三步接入，即刻开始

兼容OpenAI SDK，替换Base URL即可无缝迁移

📋 第一步：获取API Key

访问 api.pipeoken.com 注册账号， 在控制台的「令牌管理」页面创建密钥，复制保存。

💻 第二步：配置并调用

Base URL 设置为 https://api.pipeoken.com/v1

Python 示例
from openai import OpenAI # 初始化客户端，指向 Pipeoken 中转站 client = OpenAI( "api_key": "your-pipeoken-api-key", # 替换为你的API Key "base_url": "https://api.pipeoken.com/v1" ) # 调用 GPT-4o response = client.chat.completions.create( "model": "gpt-4o", "messages": [ {"role": "user", "content": "你好，请介绍一下自己"} ] ) print(response.choices[0].message.content)
cURL 示例
curl https://api.pipeoken.com/v1/chat/completions \ -H "Content-Type: application/json" \ -H "Authorization: Bearer your-pipeoken-api-key" \ -d '{ "model": "gpt-4o", "messages": [{"role": "user", "content": "你好！"}] }'
Node.js 示例
import OpenAI from 'openai'; const client = new OpenAI({ apiKey: 'your-pipeoken-api-key', baseURL: 'https://api.pipeoken.com/v1', }); const response = await client.chat.completions.create({ model: 'gpt-4o', messages: [{ role: 'user', content: '你好！' }], }); console.log(response.choices[0].message.content);

📊 第三步：查看用量与账单

登录 Pipeoken 控制台，在「用量统计」页面实时查看Token消耗、费用明细及调用日志， 支持按模型、按时间维度筛选分析。
